\# Tic Tac Toe (Unity)



A local multiplayer Tic Tac Toe game built in Unity with UI polish, animations, and game systems such as statistics, settings, and themes.



\---



\## 🎮 Game Overview



This is a classic Tic Tac Toe game designed for two players on a single device.  

It supports both portrait and landscape orientations and includes additional UI/UX features to make the experience more polished and modern.



\---



\## ✨ Features



\### Core Gameplay

\- Local multiplayer (same device)

\- Classic 3x3 Tic Tac Toe rules

\- Win, draw, and game result detection

\- Restart / replay functionality



\### UI \& Navigation

\- Main menu scene with:

&#x20; - Play button

&#x20; - Stats button

&#x20; - Settings button

&#x20; - Exit button

\- In-game result popups (win/draw)

\- Exit confirmation popup

\- Smooth UI transitions and animations



\### Settings System

\- Theme selection (visual customization)

\- Audio controls:

&#x20; - Background music toggle/volume

&#x20; - Sound effects toggle/volume



\### Statistics System

\- Tracks game results:

&#x20; - Wins (Player X / Player O)

&#x20; - Draws

&#x20; - Total games played



\### Audio \& Effects

\- Background music (BGM)

\- SFX:

&#x20; - Button click sounds

&#x20; - Piece placement sounds

&#x20; - Win / strike sound effects

&#x20; - Popup animation sounds



\### UI/UX Polish

\- Animated popups

\- Visual feedback on interactions

\- Clean responsive layout for:

&#x20; - Portrait mode

&#x20; - Landscape mode
